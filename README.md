Ein Subnetz- und IP-Rechner, gebaut mit **FastAPI** (Backend) und **React** (Frontend).

Aus einer IP-Adresse und CIDR-Notation (z.B. `192.168.1.0/24`) berechnet das Tool
Netzwerkadresse, Broadcast-Adresse, nutzbare Host-Range, Anzahl der Hosts und mehr.

## ✨ Features

- 🌐 Netzwerk- & Broadcast-Adresse aus IP + CIDR berechnen
- 📍 Erste/letzte nutzbare Host-Adresse & Anzahl der Hosts
- 🔄 Subnetzmaske ↔ CIDR umrechnen (`255.255.255.0` ↔ `/24`)
- 🔢 IP in Binär- und Dezimaldarstellung anzeigen
- ✅ Prüfen, ob eine IP in einem Netz liegt
